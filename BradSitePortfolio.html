Here is my portfolio!

Below is a collection of games, projects, and assets I've made. If you are interested in viewing the source code for any project, click the github icon in the section next to the game's image.

Dungeon Dice Frogs

Dungeon Dice Frogs is a game jam submission for the Mini Jam 106: Frogs Game Jam. I worked together with a colleague to create this 3D heaxgonal tile turn-based game where players choose frogs, who are also dice, and use them by rolling and fighting against an A.I. enemy. This game is currently still in a private repo but can be played on my itch page. The github icon for this description will redirect to the itch page. Made in Unity. Published on Itch.

One Room Dungeon

One Room Dungeon is a game jam submission for the IndieTales Game Jam. I worked alone to design, program, and create the assets for this level-based 2D shooter where the walls can be moved and move on their own. Made in Unity. Published on Itch.

Temple of Hanoi

Temple of Hanoi is a pixelated version of the Tower of Hanoi concept made into a children's game. This game includes randomization of the difficulty and provides an easy learning opportunity for young people or individuals new to using a mouse an opportunity to learn. Made in Unity. All assets created by me. Published on Itch.

Fishing Craze

Fishing Craze is an easy to grasp fishing game where players use angle and strength to throw a bobber into the ocean. Fish swim inbetween invisible nodes, and the player can pull a fish out of the water once they are touching the bobber. Made in Unity. All assets made locally.

Project Excelsior

Project Exclesior is my first attempt at creating an original game with networking and multiplayer functionality. The idea of the game is that a group of adventurers (players) are exploring a location for gold, and one adventurer is attempting to vanquish the other adventurers before the gold goal is reached. Made in Unity. Uses third-party assets.

Quest

Quest is one of my first ever Unity game creations after college. Quest is a choice driven exploration game where players are attempting to escape a cave after their village is attacked and they are captured. The game is built using Unity Scriptable objects. Made in Unity. All assets made locally.

SpaceJam

SpaceJam originally started as a simple clone of the popular Galaga game. It eventually evolved into a small "runner-like" game that I fully-developed and play on my phone from time to time. Made in Unity. All assets made by myself.

N's Game

N's game is a small 2D sidescrolling hack-n-slash game I made for a friend's birthday. He was turning 24 at the time, and on defeating the 24th enemy it shared a small happy birthday message. N's game uses third-party assets I downloaded from the asset store.
